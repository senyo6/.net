Bootstrap Guide

Download Boostrap

This boilerplate already contains the downloaded bootrstrap (currently v3.3.7) so you can go to the next step

We like to install the packages via NPM, the package manager for Node.js

You can include the Bootstrap CSS and Javascript from MaxCDN. Use these Bootstrap CDN links
Don't forget that the JS files include order matters. First is the jQuery and then comes the Boostrap (Bootstrap requires jQuery to work).

Include Bootstrap

Include you Bootstrap files in your index.html file (in our boilerplate it's already included for you in the index.html, you just have to use it )
